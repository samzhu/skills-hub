How it works

Built on events, not imports.

Every skill lifecycle moment — created, scanned, published, flagged, downloaded — is a durable domain event. That's how we keep audit trails clean and trust signals visible.

domain_events → projections → read models

Why teams choose Skills Hub

Governance without slowing anyone down.

Risk-tiered by default

Every upload is scanned for dangerous shell patterns, sensitive path access, and untrusted domains. Low-risk skills auto-publish; high-risk go to review.

Semantic search that gets intent

Describe the task — "deploy a service to kubernetes" — and get ranked matches with explanations. Powered by Gemini embeddings over your org's registry.

Full-lifecycle analytics

Downloads, ratings, version adoption, flag volume. Authors see what lands; admins see what needs attention. All derived from a single event stream.
