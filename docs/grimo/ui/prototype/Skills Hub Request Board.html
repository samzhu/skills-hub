我要開需求

寫你真的遇到的工作問題；送出後大家可以按「我也要」，技能作者可以從討論裡回應。

不需要指定負責人；需求會先進入投票列表。

送出後會發生什麼

需求先被大家看見，再用票數排優先度。

1. 出現在瀏覽需求其他人可以搜尋、留言、按「我也要」。
2. 票數推高排序「我也要」越多，需求越容易被看見。
3. 作者自願回覆有人想做時可以在 detail 頁留言協作。
