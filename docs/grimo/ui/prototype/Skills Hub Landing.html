Why Skills Hub

Governance without slowing anyone down.

The three things that make Skills Hub different from a plain GitHub repo of SKILL.md files.

Risk-tiered by default

Every upload is scanned for dangerous shell patterns, sensitive path access, and untrusted domains. Low-risk skills auto-publish; high-risk go to the review queue — without blocking the author.

Semantic search that gets intent

Describe the task — "deploy a service to kubernetes with auto-scaling" — and get ranked matches with reasoning. Powered by Gemini embeddings over your org's full registry.

Full-lifecycle analytics

Downloads, ratings, version adoption, flag volume — authors see what lands, admins see what needs attention. All derived from a single append-only event stream.
