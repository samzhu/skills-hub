When to use this skill

Invoke when working with docker-compose.yml files — authoring a new stack, adding services, debugging why containers fail to start, or hardening an existing compose file for production.

  • Scaffolding new multi-service stacks with sensible defaults
  • Adding healthchecks, depends_on conditions, and restart policies
  • Converting ad-hoc docker run commands into compose services
  • Reviewing compose files for rootless, resource-limit, and secret-handling issues
