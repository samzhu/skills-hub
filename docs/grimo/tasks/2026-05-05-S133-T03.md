## BDD（manual scenarios）

```bash
# 啟動兩 server
cd backend && ./gradlew bootRun                 # :8080 (Spring Boot, LAB mode)
cd frontend && npm run dev                       # :5173 (Vite dev proxy)
```

Scenarios（每個都記錄 PASS/FAIL + console + network log）：

### S1 — agent curl: PUBLISHED PUBLIC skill (AC-1)
```bash
SKILL_ID=$(curl -s 'http://localhost:8080/api/v1/skills?size=1' | jq -r '.content[0].id')
curl -i "http://localhost:8080/api/v1/skills/${SKILL_ID}/skill.md"
```
✓ HTTP 200
✓ `Content-Type: text/markdown`
✓ `Cache-Control: public, max-age=60` (or `max-age=60, public`)
✓ Body 開頭為 SKILL.md frontmatter `---\nname: ...`

### S2 — anonymous PRIVATE skill (AC-2)
LAB mode 下無真實 anonymous。改 OAuth mode（`SKILLSHUB_SECURITY_OAUTH_ENABLED=true`）
重啟 backend，確保有一個 PRIVATE skill 無 `*:read`：
```bash
curl -i "http://localhost:8080/api/v1/skills/${PRIVATE_ID}/skill.md"
```
✓ HTTP 401 (anonymous) 或 403 (authenticated 無 grant)
✓ Body 不含 skill name / description

### S3 — SUSPENDED skill (AC-3)
透過 admin endpoint suspend 一個 skill 後：
```bash
curl -i "http://localhost:8080/api/v1/skills/${SUSPENDED_ID}/skill.md"
```
✓ HTTP 403
✓ ErrorResponse `error: "SKILL_SUSPENDED"`

### S4 — non-existent skill (AC-4)
```bash
curl -i "http://localhost:8080/api/v1/skills/00000000-0000-0000-0000-000000000000/skill.md"
```
✓ HTTP 404

### S5 — UI dropdown shows on PUBLISHED (AC-5)
- Browser open `http://localhost:5173/skills/${SKILL_ID}`（PUBLISHED）
- ✓ SkillHero 右側 button cluster 出現「Markdown ▾」trigger
- ✓ Click trigger → menu 展開 2 items

### S6 — UI dropdown hidden on DRAFT / SUSPENDED (AC-6)
- Browser open DRAFT skill 詳情頁
- ✓「Markdown ▾」不存在 (DOM query)
- 同上對 SUSPENDED

### S7 — Copy as Markdown 實際寫入剪貼簿 (AC-7)
- Browser open PUBLISHED skill
- DevTools open Network + Console
- Click「Markdown ▾」→ ✓ Network 出現 `GET /skill.md` (prefetch)
- Click「複製為 Markdown」→
  ✓ sonner toast 出現「已複製到剪貼簿」
  ✓ Browser 切 textarea 貼上 → 內容 = SKILL.md raw body
  ✓ Console 無 `NotAllowedError`
- **Safari 必測**（Chrome lenient 不能代表 Safari 行為）：
  ✓ Safari macOS 同樣 PASS

### S8 — Open Markdown 新分頁 (AC-8)
- Click「開啟 Markdown」
- ✓ 新分頁開啟，URL = `/api/v1/skills/{id}/skill.md`
- ✓ 顯示 raw markdown text（瀏覽器 default text/markdown 渲染或 plain text）

## Boundary scenarios（push known constraints）

- **S9 — large SKILL.md**: 找一個 SKILL.md > 100KB 的 skill，Copy 走 ClipboardItem 應仍 OK（無 size error）
- **S10 — special chars**: SKILL.md 含中文 / emoji / code blocks `\`\`\`` → 複製內容必須 UTF-8 完整保留
