# S142a-T05: Sidebar (8 cards: 5 always + 3 contextual)

## Spec
S142a — SkillDetailPage v2 Frontend Rework

## BDD

Given Sidebar container
When  渲染
Then  顯 5 always-visible cards in order:
  1. InstallCard
  2. SparklineCard (移自 hero — per S098e original location)
  3. DetailsCard (Published / License / Size / Files / Scripts)
  4. CompatibilityCard (chips)
  5. VersionHistoryMini (latest 4)
And   contextual cards 依 activeTab 顯/隱：
  - activeTab === 'quality' → 加 QualityTOCCard + ReproducibilityCard
  - activeTab === 'security' → 加 SecurityAuditCard
  - 其他 tab → 兩種 contextual 都隱藏
And   sidebar layout: flex column + gap 14px + border-left .5px var(--line) + padding-left 22px

Given InstallCard
When  user 點複製按鈕
Then  navigator.clipboard.writeText("skills-hub install {author}/{name}") 被呼叫
And   按鈕 icon 切換 checkmark；1500ms 後還原 (reuse Tessl 風格 affordance)

Given InstallCard
Then  顯 dropdown「CLI ▼」(static label，無下拉行為 — 對齊 D7 default)
And   命令文字 mono font + 黑底 rounded
And   底部「What are skills?」link → /docs/your-first-skill (S094d)

Given SparklineCard
Then  顯 30d trend (reuse Sparkline component from S096d3)
And   下方 7d / 30d 累計數字
And   無 data 時顯「—」(per §B.1 default)

Given DetailsCard
Then  顯 5 行 meta：
  - Published: latestVersionPublishedAt (從 Skill response 取 — S142b)
  - License: license (從 Skill response — S142b)
  - Size: fileSize (從 latest SkillVersion — 既有)
  - Files: fileCount (從 latest SkillVersion — 既有)
  - Scripts: 「None」如 fileCount(scripts/) === 0；否則「N scripts」 (frontend derive from /files endpoint OR backend supplement future spec)

Given CompatibilityCard
Then  顯 chips wrap (從 skill.compatibility — S142b)
And   chip 樣式對齊 prototype `.chip`
And   8+ runtimes flex-wrap 折行 (per §B.1 default — 不做 +N more)

Given VersionHistoryMini
Then  顯 latest 4 versions
And   每 row: ver number (mono) + relative time + 下載 icon
And   第一筆加「latest」badge
And   下方「查看全部 →」link，click → 切到 Versions tab (per §B.1 default)

Given QualityTOCCard (Quality tab only)
Then  顯 Table of contents:
  - 01 Validation
  - 02 Implementation
  - 03 Discovery
And   click → smooth scroll to corresponding QualityTabV2 section

Given ReproducibilityCard (Quality tab only)
Then  顯 evaluator metadata (從 ScoreResponse meta block):
  - Evaluated: relative time
  - Evaluator: gemini-2.5-flash (mono)
  - Prompt ver: v1 (mono)
  - Skill ver: v2.1.0 (mono)

Given SecurityAuditCard (Security tab only)
Then  顯 audit metadata (從 SecurityReport):
  - Last scanned: relative time
  - Engine: risk-scanner v1.0 (mono)
  - Rule set: 2026-05 (mono)
  - Dep scan: 「not yet enabled」placeholder (until dep scanning integrated)
