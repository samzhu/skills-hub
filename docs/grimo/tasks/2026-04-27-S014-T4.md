### 2. SkillQueryService.getCategoryCounts — Aggregation → SQL GROUP BY

```java
public List<CategoryCount> getCategoryCounts() {
    return jdbc.query("""
        SELECT category AS name, COUNT(*) AS count
          FROM skills
         WHERE category IS NOT NULL
         GROUP BY category
         ORDER BY count DESC
        """, (rs, rowNum) -> new CategoryCount(
                rs.getString("name"),
                rs.getLong("count")
        ));
}
```

> 也可考慮 `@Query` on `SkillReadModelRepository`，但因 `CategoryCount` 是 view DTO（非 entity），用 JdbcTemplate + RowMapper 更直觀。

### 3. SkillReadModelRepository — 加 incrementDownloadCount

```java
public interface SkillReadModelRepository extends ListCrudRepository<SkillReadModel, String> {

    @Modifying
    @Query("UPDATE skills SET download_count = download_count + 1, updated_at = :ts WHERE id = :id")
    int incrementDownloadCount(@Param("id") String id, @Param("ts") Instant ts);
}
```

### 4. SkillProjection.on(SkillDownloadedEvent) — read-modify-write → atomic

```java
// Before
@EventListener
void on(SkillDownloadedEvent event) {
    repo.findById(event.aggregateId()).ifPresent(existing -> {
        var updated = new SkillReadModel(..., existing.downloadCount() + 1, ...);
        repo.save(updated);
    });
}

// After
@EventListener
void on(SkillDownloadedEvent event) {
    repo.incrementDownloadCount(event.aggregateId(), Instant.now());
    log.atDebug().addKeyValue("skillId", event.aggregateId()).log("投影已累加下載次數");
}
```

### 6. AtomicDownloadCountTest — 新增

```java
@SpringBootTest
@Testcontainers
class AtomicDownloadCountTest {

    @Autowired SkillReadModelRepository repo;
    @Autowired SkillProjection projection;

    @Test
    @DisplayName("AC-6: 100 次並發 SkillDownloadedEvent 累加為 100（atomic）")
    @Tag("AC-6")
    void concurrentIncrement_noLostUpdate() throws Exception {
        var skillId = UUID.randomUUID().toString();
        repo.save(buildSkill(skillId, 0L));

        var executor = Executors.newFixedThreadPool(10);
        var latch = new CountDownLatch(100);

        for (int i = 0; i < 100; i++) {
            executor.submit(() -> {
                try {
                    projection.on(new SkillDownloadedEvent(skillId, "1.0.0"));
                } finally {
                    latch.countDown();
                }
            });
        }
        latch.await(30, TimeUnit.SECONDS);
        executor.shutdown();

        var final_ = repo.findById(skillId).orElseThrow();
        assertThat(final_.downloadCount()).isEqualTo(100L);
    }
}
```
