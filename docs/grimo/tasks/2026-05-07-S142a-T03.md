## Spec

## BDD

Given skill 含 frontmatter + body markdown
When  SkillMdTab 渲染
Then  panel 顯 frontmatter `<pre>` block + tok-key / tok-str / tok-num / tok-kw token classes
And   description value 用 accent-text (#C9C5F2) 紫色高亮 (per 工程說明 §6 — AI embedding 焦點)
And   --- delimiters 用 ink-3 灰色 + opacity 0.5
And   下方接 H1 + H2 sections markdown render (reuse 既有 markdown library)

Given GET /scores 200
When  QualityTabV2 渲染
Then  panel 顯 3 section: Validation / Implementation / Discovery (對齊中文 axis label：規格驗證 / 實作品質 / 觸發能力 per S135b)
And   每 section head: title + sub-label + score num + accent 漸層 progress bar
And   每 section 含 table (Dim / Reasoning / Score)
And   table row score dot 顏色：accent (full) / amber (>= 60%) / red (< 60%) per 工程說明 §7 (使用 ScoreDot from T02 shared)
And   reasoning 預設展開；Show less / Show more 切換 collapse

Given Quality 404
When  QualityTabV2 渲染
Then  panel 顯「此版本尚未評分」(reuse S135b QualityTab fallback pattern)

Given GET /versions 回 N 筆
When  VersionsTabV2 渲染
Then  panel 顯 N 個 changelog-style cards
And   每 card: ver number (mono font) + latest badge (only 第一筆) + relative time + changelog 描述
And   description 3-line clamp + Show more
And   無 latestVersion link (Sidebar 有 mini list 走 sidebar Path)

Given GET /security-report 200 + overall=pass
When  SecurityTab 渲染
Then  hero shield: stroke=#1D9E75, fill=rgba(29,158,117,0.15), green checkmark icon
And   hero title「No security issues found」(綠色)
And   hero sub「Scanned N days ago · engine v1.0 · rule set 2026-05」
And   下方 4-quad cards (Shell / Paths / Secrets / Deps)
And   每 quad: seg color + name + status badge (✓ Passed / ! Review / ✗ Fail) + description (從 detail string)

Given securityReport.overall === "warn"
When  SecurityTab 渲染
Then  hero shield amber (stroke=#EF9F27, fill=rgba(239,159,39,0.12), warning icon)
And   hero title「N issue requires review」(amber)

Given securityReport.overall === "fail"
When  SecurityTab 渲染
Then  hero shield red (stroke=#E24B4A, fill=rgba(226,75,74,0.12), X icon)
And   hero title「N issues require attention」(red)

Given 404 SECURITY_NOT_SCANNED
When  SecurityTab 渲染
Then  panel 顯「Security report 尚未掃描」fallback

Given Reviews tab + Flags tab (既有 ReviewsPanel S098e2 / FlagsList S112)
When  S142a 不重做這兩 tab
Then  維持既有 components 原樣 import；只改 tab order 對齊 v2 prototype 的位置
