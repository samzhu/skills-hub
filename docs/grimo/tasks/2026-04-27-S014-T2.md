## 工作明細

### 1. DomainEvent.java 改寫

```java
// Before
@Document("domain_events")
@CompoundIndex(def = "{'aggregateId': 1, 'sequence': 1}", name = "...")
public record DomainEvent(
    @Id String id, ..., Map<String, String> metadata
) {}

// After
@Table("domain_events")
public record DomainEvent(
    @Id String id,
    @Column("aggregate_id") String aggregateId,
    @Column("aggregate_type") String aggregateType,
    @Column("event_type") String eventType,
    @Column("payload") Map<String, Object> payload,
    @Column("sequence") long sequence,
    @Column("occurred_at") Instant occurredAt,
    @Column("metadata") Map<String, Object> metadata    // 升型別 String → Object
) {}
```

> `metadata` 型別提升 `Map<String,String> → Map<String,Object>` — 既有呼叫端都用泛型 erased type，向下相容；Converter 也無法只針對 `<String,String>` 註冊（會與 `<String,Object>` 衝突）。

### 2. DomainEventRepository.java 改寫

```java
// Before
public interface DomainEventRepository extends MongoRepository<DomainEvent, String> {
    List<DomainEvent> findByAggregateIdOrderBySequenceAsc(String aggregateId);
    Optional<DomainEvent> findTopByAggregateIdOrderBySequenceDesc(String aggregateId);
}

// After
public interface DomainEventRepository extends ListCrudRepository<DomainEvent, String> {
    List<DomainEvent> findByAggregateIdOrderBySequenceAsc(String aggregateId);
    Optional<DomainEvent> findTopByAggregateIdOrderBySequenceDesc(String aggregateId);
}
```

> Spring Data JDBC 不需要 `MongoRepository` 的繼承層；`ListCrudRepository` 提供 save/findById/findAll/deleteById/count，而 derived query method 命名規則一致。

### 4. 新增 DomainEventSequenceUniquenessTest

```java
@DataJdbcTest
@Testcontainers
class DomainEventSequenceUniquenessTest {

    @Autowired DomainEventRepository repo;

    @Test
    @DisplayName("AC-3: (aggregate_id, sequence) UNIQUE 約束防止重複 sequence")
    @Tag("AC-3")
    void uniqueConstraint_blocksDuplicateSequence() {
        var aid = UUID.randomUUID().toString();
        repo.save(buildEvent(aid, 1));
        repo.save(buildEvent(aid, 2));

        assertThatThrownBy(() -> repo.save(buildEvent(aid, 2)))
                .isInstanceOf(DataIntegrityViolationException.class);
    }

    @Test
    @DisplayName("AC-3: findByAggregateIdOrderBySequenceAsc 回傳 sequence 升冪排列")
    @Tag("AC-3")
    void replay_returnsEventsInSequenceOrder() {
        var aid = UUID.randomUUID().toString();
        // 故意亂序寫入
        repo.save(buildEvent(aid, 3));
        repo.save(buildEvent(aid, 1));
        repo.save(buildEvent(aid, 2));

        var events = repo.findByAggregateIdOrderBySequenceAsc(aid);
        assertThat(events).hasSize(3);
        assertThat(events).extracting(DomainEvent::sequence).containsExactly(1L, 2L, 3L);
    }
}
```
