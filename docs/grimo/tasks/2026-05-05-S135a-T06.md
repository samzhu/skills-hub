# S135a-T06: QualityScoreController GET Endpoint

## Spec
S135a — Backend Quality Score（read API；GET /api/v1/skills/{id}/scores 回 3-axis breakdown + total；未評分 404 with QUALITY_NOT_EVALUATED）

## BDD

```
Given skill `demo-skill` 已評分（skill_scores 表 3 rows）
When GET /api/v1/skills/{id}/scores
Then 200 OK
And response body 含 keys: skillId, skillVersionId, skillVersion, evaluatedAt, evaluatorVersion,
                          validation, implementation, activation, total
And 每 axis 含 totalScore (0-100, integer) + dimensions (object)
And total = round(0.2 × validation.totalScore + 0.4 × implementation.totalScore + 0.4 × activation.totalScore)

Given skill `demo-skill` 已評多版本（v1.0.0 + v1.1.0），querystring ?versionId={v1.0.0 id}
When GET /api/v1/skills/{id}/scores?versionId={v1.0.0 id}
Then return v1.0.0 那次評分（不是 latest）
And response 含 skillVersion = "1.0.0"

Given skill `pending-skill` v1.0.0 剛 publish，listener 還沒跑完
When GET /api/v1/skills/{id}/scores
Then 404 Not Found
And response.error == "QUALITY_NOT_EVALUATED"
And response.message == "Score will be available shortly after publish."

Given skill id 不存在
When GET /api/v1/skills/{nonExistentId}/scores
Then 404（同 QUALITY_NOT_EVALUATED — 不需區分 skill 不存在 vs 未評分；對 caller 結果一致）

Given skill PRIVATE 無 ACL grant
When unauthorized user GET
Then 401 / 403（per 既有 SecurityConfig + @PreAuthorize；對齊 SkillQueryController.getById S122 pattern）
```

## Implementation Notes

- 新 `QualityScoreController.java`：spec §4.2 code skeleton
  - `@GetMapping` `/api/v1/skills/{skillId}/scores` + 可選 `@RequestParam(required = false) String versionId`
  - `@PreAuthorize("hasPermission(#skillId, 'Skill', 'read')")` — 對齊 S122 既驗 pattern
  - 邏輯：先 alread-evaluated check via repo（findLatestBySkillVersionId or findLatestBySkillId）
    - 若 rows.isEmpty() → throw `NotEvaluatedException(skillId)` → GlobalExceptionHandler map 為 404 `QUALITY_NOT_EVALUATED`
    - 否則組 ScoreResponse with total formula
- 新 `ScoreResponse.java` (record DTO)：spec §4.3 response shape
  - 含 nested record `AxisScore(int totalScore, Map<String, Object> dimensions)`
  - `static ScoreResponse from(List<SkillScore> rows)`：把 3 rows 轉成 axis-keyed structure + 計 total
- 新 `NotEvaluatedException.java`：extends RuntimeException；by `GlobalExceptionHandler` map 到 404 with error code `QUALITY_NOT_EVALUATED`
  - 加進 `shared/api/GlobalExceptionHandler.java`（既有 — per ErrorResponse pattern）@ExceptionHandler
- Test：`QualityScoreControllerTest extends WebMvcSliceTestBase`（per S025b — `@WebMvcTest(QualityScoreController.class)` slice）
  - `@MockitoBean SkillScoreRepository scoreRepo`（controller-specific dep；slice 不 wire repo 真實）
  - 用 `.with(jwt())` post-processor 模擬 authenticated（per WebMvcSliceTestBase pattern）
  - 5 個 test cover 5 個 BDD scenario
- AC tag：`@DisplayName("AC-S135a-3: ...")` / `AC-S135a-4`
