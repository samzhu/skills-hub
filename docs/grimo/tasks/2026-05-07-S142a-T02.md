# S142a-T02: PageHeader + Hero region (3 cards + StatStrip + click-to-tab)

## Spec
S142a — SkillDetailPage v2 Frontend Rework

## BDD

Given skill data with `verified=true`
When  PageHeader 渲染
Then  顯 IconTile + skill-name + ver-badge + risk-pill + verified-pill (綠勾 svg + "Verified")
And   skill-desc (line-clamp-2 + "展開" if > 200 字 per §B.1 default)
And   meta-row: by author + license + Updated N days ago + category
And   actions: Star button (lucide Star icon, click=Subscribe behavior per AC-3) + Download CTA (BorderBeam wrap)

Given skill.verified === false
Then  PageHeader 不渲染 verified-pill (純隱藏)

Given Star button + 已登入 + 非 author
When  click Star
Then  toggle subscribe / unsubscribe (reuse 既有 useIsSubscribed / useSubscribeSkill / useUnsubscribeSkill)
And   icon = ⭐ (lucide Star icon)；既有 PageHeader 鈴鐺 icon 拿掉

Given GET /scores 200 with skillScore=89
When  SkillScoreBadge 渲染
Then  SVG hexagon ring 出現 (r=50, viewBox 120x120)
And   arc fill stroke-dashoffset = 314.16 × (1 - 89/100) + 26.2 = 60.76
And   inner text "SKILL SCORE" / "89" / "/100" 各依 SVG <text> 渲染
And   transform="rotate(135 60 60)" → arc 從左下開始，順時針 270°

Given GET /scores 404 QUALITY_NOT_EVALUATED
When  SkillScoreBadge 渲染
Then  hexagon arc 顯灰色 (track 不填 fill arc)
And   center 顯「—」(代替數字)
And   下方小字「評分計算中」

Given 已評分 + 已掃描 skill
When  HeroMetricsRow 渲染
Then  grid-template-columns: 160px 1fr 1fr
And   QualityHeroCard 顯 92% + Validation 100 / Implementation 85 / Discovery 92 breakdown + accent 漸層 progress bar
And   SecurityHeroCard 顯「Passed」+ 4-segment bar (4 個 #1D9E75→#6FD8B0 漸層 segments) + Shell·Paths·Secrets·Deps breakdown

Given securityReport.checks.paths.status === "warn"
When  SecurityHeroCard 渲染
Then  4-segment bar 第 2 段 (paths) 變 amber，其他 3 段保持綠色
And   hero card border 加 rgba(239,159,39,.3)
And   hero value 顯「1 Issue」+ sub「Sensitive path access detected」(從 detail derive)

Given QualityCard click
When  user 點 hero Quality card
Then  Quality tab active
And   QualityCard 加 border-color: rgba(127,119,221,.45) active 樣式
And   Sidebar 顯 TableOfContents + Reproducibility cards (其他 contextual hidden)
(類似行為 SecurityCard click → Security tab + AuditMetadata sidebar)

Given GET /stats 14d array + GET /skills/{id} 含 averageRating / reviewCount / versionCount / openFlagCount
When  StatStrip 渲染
Then  4 cells: Downloads / Rating / Versions / Open flags
And   Downloads cell 含「↑ N%」(綠) 或「↓ N%」(紅) — frontend derive: (thisWeek - lastWeek) / lastWeek × 100
And   Rating cell 顯「{averageRating} / 5 · {reviewCount} reviews」
And   Versions cell 顯「{versionCount}」
And   Open flags cell 顯「{openFlagCount}」；> 0 時數字紅色

Given Quality 整體分數 < 60 時 hero 漸層 (per §B.2 default)
Then  仍用 accent 漸層 (不能跟風險語義色混用)
And   數字 label 用 amber/red text
