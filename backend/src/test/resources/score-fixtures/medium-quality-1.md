# Code Reviewer

This skill reviews code and provides feedback on quality, readability, and potential bugs.

## What it does

The code reviewer examines code and provides feedback on:
- Code quality
- Potential bugs
- Performance issues
- Best practices

## How to use

To use this skill, provide the code you want reviewed. The reviewer will analyze it and give you feedback.

You can ask it to focus on specific aspects like security or performance.

## Tips

- Provide the full file for better context
- Mention the language if it's not obvious
- Tell the reviewer what kind of feedback you want

The reviewer will look at the code and tell you what improvements could be made.
